# Hello

### 关于这个项目的一些说明
+ 我将每一章节的依赖包都放在了外面,不然每一节都需要安装一遍,很麻烦。
+ 使用`yarn`代替`npm`安装包依赖,因为`yarn`比较快。
+ 尽量使用`ES6`的语法。
